### HTML BASICS  

Minimum html required for a page:  
```
<!DOCTYPE html>
<html>
  <head>
    <title>Website Title</title>
  </head>
  <body>

  </body>
</html>
```

Recommended html for a page:  
```
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>
```

Some common meta tags:  
```
<!-- Character set to use -->
<meta charset="UTF-8">

<!-- Viewport is important for responsive design -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Compatibility for older browsers such as ie -->
<meta http-equiv="X-UA-Compatible" content="ie=edge">

<!-- Meta tag to give info for search engines -->
<meta name="description" content="Description of what website is about">

<!-- Meta tag for keywords for website, keep to around 8-12 words max -->
<meta name="keywords" content="drones, drone flying, mavic, ariel photography, drone pilot">

<!-- Only use following if dont want website to be searched and indexed -->
<meta name="robots" content="NOINDEX, NOFOLLOW">
```
